Summation

Notation

element meaning default value
f(e) the function we transform the element with e
cond(e) the condition predicate function true for all e
op(a, b) the operation a + b
identity the identity where a + identity = a 0
s sum; the result N/A
t the “traversable”, or enumerable we iterate on N/A
e the current element N/A

Pseudocode

s := identity
for e in t:
    if cond(e):
        s := op(f(e), s)

Example: Simple Addition

element value
cond(e) true for all e
f(e) e
op(a, b) a + b
identity 0 
SUMMATION(t,
          f(e) -> e,
          cond(e) -> true,
          op(a, b) -> a + b,
          identity -> 0)

You don’t need to specify the parts that have default value. Also, usually you do something with the result, like assign it to a variable: 

SUMMATION(e)

Example: Product

element value
cond(e) true for all e
f(e) e
op(a, b) a * b
identity 1 
SUMMATION(t,
          f(e) -> e,
          cond(e) -> true,
          op(a, b) -> a * b,
          identity -> 1)

You don’t need to specify the parts that have default value: 

SUMMATION(t,
          op(a, b) -> a * b,
          identity -> 1)

Example: Sum of Even Squares

Sum the squares of the even numbers.

element value
cond(e) e % 2 = 0
f(e) e2
op(a, b) a + b
identity 0 
SUMMATION(t,
          f(e) -> e^2,
          cond(e) -> e % 2 = 0)

Example: Selecting Even Numbers

Selecting all the even numbers and adding it to a sequence. We’re replacing addition with concatenation, so instead of having a sum of the even numbers, we’re going to have a list of them. We’re having f(e) -> [e] because we can only concatenate sequences together, so we need one-element sequences.

element value
f(e) [e]
cond(e) e % 2 = 0
op(a, b) concatenate(a, b)
identity empty sequence []

concatenate(a, b) can be written also as \(a ++ b\). For example \([1, 6] ++ [2, 4] = [1, 6, 2, 4]\). 

SUMMATION(t,
          f(e) -> [e]
          cond(e) -> e % 2 = 0,
          op(a, b) -> a ++ b,
          identity -> [])

Counting

Notation

element meaning default value
cond(e) the condition predicate function always specified
c count; the result N/A
t the “traversable”, or enumerable we iterate on N/A
e the current element N/A

Pseudocode

c := 0
for e in t:
    if cond(e):
        c := c + 1

Example: Counting Even Numbers

element value
cond(e) e % 2 = 0 
COUNTING(t, cond(e) -> e % 2 = 0)

Linear Search

Notation

element meaning default value
cond(e) what we are looking for, the condition always specified
l boolean, true if we found the element N/A
elem the elment we found, null if not found N/A
t the “traversable”, or enumerable we iterate on N/A
e the current element N/A

Pseudocode

Using l: 

l := false
for e in t:
    if cond(e):
        l := true
        elem := e
        break

Using a nullable of optional type instead of l. Here, elem is null if we have not found what we were looking for. 

elem := null
for e in t:
    if cond(e):
        elem := e
        break

Example: Looking for 9

element value
cond(e) e = 9
    
SEARCH(t, cond(e) -> e = 9)

Maximum Selection

Notation

element meaning default value
f(e) the function we transform the element with e
max the maximum value N/A
elem the element with maximum value N/A
t the “traversable”, or enumerable we iterate on N/A
e the current element N/A

Pseudocode

The precondition is that there is at least one element in \(t\). When implementing in Java, we should check this. We denote the part of the sequence that doesn’t include the first element by t[1:]. We could also just use t and have one unnecessary comparison. 

max := t[0]
for e in t[1:]:
  if f(e) > max:
    max := f(e)
    elem := e

Example

MAX(t, f(e) -> e)

Conditional Maximum Search

Notation

element meaning default value
f(e) the function we transform the element with e
cond(e) we find the maximum of elements where cond is true always specified
l true if we found the maximum N/A
max the maximum value N/A
elem the element with maximum value N/A
t the “traversable”, or enumerable we iterate on N/A
e the current element N/A

Pseudocode

The value of l will be false until we find an element for which cond(e) is true. When we find the first such element, we set l to true, and (elem, max) to that element. 

l := false
for e in t:
  if cond(e):
    if not l:
      l := true
      elem := e
      max := f(e)
    else if max < f(e):
      elem := e
      max := f(e)

Example: Finding the Largest Even Number

CONDMAX(t, f(e) -> e, cond(e) -> e % 2 = 0)
