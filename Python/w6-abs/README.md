# Week 6

## Overview

Week 6 introduces object-oriented programming, package organization, and exception-oriented CLI and file tasks.

## Learning Goals

- Model real entities with classes and methods
- Understand composition vs. inheritance
- Organize code into modules/packages and entry points
- Apply robust error handling with `try/except`
