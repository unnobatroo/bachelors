# Week 2

## Overview

Week 2 focuses on the IPO model (Input -> Process -> Output), numeric algorithms, and basic control flow.

## Learning Goals

- Distinguish input, processing, and output stages in a program
- Apply loops and conditionals to solve arithmetic problems
- Practice writing step-by-step algorithmic solutions

Solutions in this week emphasize decomposition into small stages and avoiding unnecessary work (for example, scanning divisors only up to the square root).
