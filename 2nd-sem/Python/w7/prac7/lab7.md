
Implement a simple **Student Registration system**.

The application should manage student records with the following information:

* neptun code
* name
* age
* GPA
* classes 

It should support these basic operations:

* create a student
* view student information
* update student information
* delete a student

Student data should be loaded when the program starts, so previously saved records are available.

Requirements:

* validate all user inputs
* handle possible errors properly
* organize the program into separate modules
